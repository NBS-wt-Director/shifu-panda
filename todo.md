## 📋 ОБЩЕЕ ОПИСАНИЕ ПРОЕКТА

Веб-сайт для центра функционального развития "Шифу Панда" (г. Екатеринбург). Сайт включает публичную часть для посетителей и административную панель для управления контентом.

## 🌐 ПУБЛИЧНЫЕ СТРАНИЦЫ

### 1. Главная страница (`/`)
**Файл:** `src/app/page.tsx`

**Описание:** Основная страница сайта с последовательным отображением секций. Поддерживает параметр `?scrollTo=section-id` для скролла к разделу.

**Компоненты:**
- `SiteHeader` - шапка сайта с логотипом, меню, заголовком страницы, кнопкой заказа звонка
- `HomeSlider` - слайдер с изображениями и заголовками
- `SectionSpacer` - разделитель между секциями (настраиваемый)
- `HomeSchedule` - секция расписания занятий
- `HomePrices` - секция с ценами
- `HomePrograms` - секция программ тренировок
- `HomeTrainers` - секция тренеров
- `HomeNews` - секция новостей
- `HomeContacts` - секция контактов
- `FullScreenImageModal` - модальное окно для просмотра изображений
- `CallModal` - модальное окно заказа звонка
- `StatsCollector` - сбор статистики посещений

**Особенности:**
- Секции отображаются в порядке, настраиваемом в админке
- Разделитель между всеми секциями единый и настраивается в админке
- Фото в карточках тренеров и программ не обрезаются (object-contain)
- Описание убрано из карточек программ и тренеров на главной
- Поддержка скролла к разделу через URL параметр

**Статус:** ✅ Выполнено

## ⚙️ НАСТРОЙКИ ХЕДЕРА

### Параметры (headerSettings в db.json)

| Параметр | Тип | Описание |
|----------|-----|----------|
| `titleSuffix` | string | Суффикс для тега title (добавляется к заголовку страницы) |
| `componentsEnabled` | object | Включение/отключение компонентов: callButton, pageTitle, menu |
| `componentsOrder` | array | Порядок компонентов: logo, pageTitle, callButton, menu |
| `homeMenuEnabled` | boolean | Показывать меню разделов главной страницы |
| `logoAnimation` | string | Анимация логотипа: none, pulse, bounce, rotate |
| `secondLineText` | string | Текст второй строки под заголовком |
| `secondLineAnimation` | string | Анимация второй строки: none, fade, slide, typewriter |

### Анимации логотипа
- **none** - без анимации
- **pulse** - пульсация (масштабирование)
- **bounce** - подпрыгивание (движение вверх-вниз)
- **rotate** - вращение

### Анимации второй строки
- **none** - без анимации
- **fade** - плавное появление
- **slide** - скольжение сверху
- **typewriter** - эффект печатной машинки
